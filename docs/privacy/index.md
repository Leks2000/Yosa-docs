
# Политика конфиденциальности приложения «Йося»

Дата вступления в силу: 19.05.2026.

## 1. Какие данные мы собираем
Приложение «Йося» (далее — «Приложение») работает преимущественно
локально. Мы собираем минимальный объём данных:

- **Анонимный device_id** (случайно сгенерированный, без привязки
  к личности) — для подсчёта DAU и стабильности приложения.
- **События аналитики** (открытие экранов, добавление блюд, покупки) —
  без содержимого ваших записей и без PII.
- **Записи о еде** хранятся ЛОКАЛЬНО на вашем устройстве (Hive).
  Мы НЕ загружаем их на сервер.
- **Текст запроса при анализе еды** временно передаётся в наш Edge
  Function для расчёта КБЖУ и НЕ сохраняется на сервере дольше времени
  обработки запроса (≤30 секунд).

## 2. Платежи
Оплата подписки Pro обрабатывается через ЮKassa (АО «НКО ЮMoney»).
Данные банковской карты НЕ передаются нам — они напрямую идут в ЮKassa.

## 3. Передача третьим лицам
Мы НЕ продаём и НЕ передаём ваши данные третьим лицам, кроме:
- ЮKassa — для обработки платежей
- Google AI (Gemini API) — текст блюда для анализа КБЖУ

## 4. Хранение
Локальные данные (журнал питания, цели, бейджи) хранятся на вашем
устройстве и удаляются вместе с удалением приложения.

## 5. Ваши права
Вы можете:
- Удалить приложение → все локальные данные стираются
- Запросить удаление учётной записи: dextyprod@mail.ru
